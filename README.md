# Let Everyone In — Initial Site (Archived)

> ⚠️ **This repository is no longer the live site.** It has been superseded by [`steadfast-a11y/leteveryonein`](https://github.com/steadfast-a11y/leteveryonein).

## What this was

This was the original website for **Steadfast Accessibility LLC** / **Let Everyone In**, built prior to a comprehensive accessibility audit and redesign. It was built with Next.js and deployed to Cloudflare Pages.

## Why it was replaced

The site underwent a full rebuild to better reflect the brand's commitment to accessibility — it was rebuilt from the ground up using a modern Vite + React stack via [Lovable](https://lovable.dev), with updated copy, design, and structure.

## Current live site

The live site is now at [leteveryonein.com](https://leteveryonein.com), sourced from [`steadfast-a11y/leteveryonein`](https://github.com/steadfast-a11y/leteveryonein).
